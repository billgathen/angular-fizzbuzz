Thanks for trying FizzBuzz!

FizzBuzz is a counting game. The rules are simple:

Output a set of numbers (say, 1-15), but whenever a number is divisible by 3, you output “Fizz” instead.
If it’s divisible by 5, you output “Buzz” instead. If it’s divisible by 3 and 5, you output “FizzBuzz” instead.
